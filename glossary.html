Tips for using the website

  • Any of the icons above can be pasted into the fighter name search to filter the fighters that meet said condition
  • Regular expression can be used in the fighter name search. For example, the vertical separator “|” can be used between names like so: “islam|volkanovski|mcgregor”.
  • The rank columns have 3 hidden values that can be used to filter the fighters: -1 for champions, 0 for interim champions, and 16 for unranked.
  • Ranges can be used to filter all numeric columns. For example, “20-29” in age to filter everyone in their 20s. Or, “-1-15” to find everyone ranked in the rank column.
